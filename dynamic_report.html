We were interested in seeing how much Instagram advertisements impact people's purchases because we wanted to analyse what specific aspects of these ads influenced people to click, and also buy. The data collected may be useful to advertisement companies, clothing stores, and anyone interested in consumer behaviour. ```{r data_and_analysis} # Read data logged_data <- read_csv("https://docs.google.com/spreadsheets/d/e/2PACX-1vSx9vbbR6YR9Fs92i5d24k-_pvkhAP327ZHxeP83Cjuooryk68WACEg8LMj-doSc-cDUSIBSN7eBAl4/pub?output=csv") # Rename variables latest_data <- logged_data %>% rename( screentime = `average daily screen time on Instagram (in hours)`, saw_ad = `Did you see an advertisement for a clothing website on Instagram today?`, more_than_one_ad = `more than one ad seen`, clicked = `influenced to click on website`, purchase = `Did you purchase anything from the website as a result of the advertisement?` ) # Summary 1: Mean screen time mean_screentime <- round(mean(latest_data$screentime, na.rm = TRUE), 1) # Summary 2: Number of participants who clicked an ad clicked_count <- sum(latest_data$clicked == "Yes") # Summary 3: Number of participants who clicked AND purchased clicked_and_purchased <- sum(latest_data$clicked == "Yes" & latest_data$purchase == "Yes") # Bar chart 1: ad exposure ad_exposure_plot <- latest_data %>% ggplot(aes(x = saw_ad)) + geom_bar() + labs( title = "Did participants see ads for clothing websites scrolling on Instagram?", x = "Saw advertisement", y = "Count" ) # Bar chart 2: clicked vs purchased clicked_vs_purchased_plot <- latest_data %>% ggplot(aes(x = clicked, fill = purchase)) + geom_bar(position = "dodge") + labs( title = "Relationship Between Clicking Ads and Purchasing", x = "Clicked on Ad", y = "Count", fill = "Purchased" ) ``` ### Summary Values (Dynamic Text) - The average daily Instagram screen time is **`r mean_screentime` hours**. - Out of `r nrow(latest_data)` participants, **`r clicked_count` clicked on an advertisement**. - Out of 20 participants, **`r clicked_and_purchased` participants both clicked on an advertisement and made a purchase**. ### Plots ```{r show_plots} ad_exposure_plot clicked_vs_purchased_plot ```
